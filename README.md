# Student Jobs Portal

## Description

Student Jobs Portal is a web and mobile platform designed to help students find part-time job opportunities and apply for them easily.
The system allows students to browse available jobs, apply for positions, and manage their applications.
Admins can manage job postings and monitor applications through an admin dashboard.

## Features

### Authentication

* User Login
* User Registration
* Secure Authentication System

### Student Features

* Student Dashboard
* Student Profile Management
* Browse Available Jobs
* Apply for Jobs
* View My Applications
* Search for Jobs
* Like Jobs
* Comment on Jobs

### Admin Features

* Admin Dashboard
* Admin Profile
* Manage Job Posts
* Monitor Applications

## Technologies Used

* **Frontend:** Web Application
* **Backend:** Firebase
* **Database:** Firestore
* **Authentication:** Firebase Authentication
